# Check in 用户操作手册

## 1. 首次安装

1. 安装并打开 `Check in`，保持右上角服务开关开启。
2. 在“系统体检”中逐项处理橙色 `SETUP`。
3. 开启 `Check in 自动打开服务`、通知、全屏提醒和精确闹钟。
4. 将电池策略设为无限制。
5. 在手机厂商设置中允许自启动、后台运行和后台弹出界面。

小米 15 / HyperOS 用户还应在最近任务中锁定应用。HyperOS 升级后可能
重置部分权限，升级后请重新检查一次。

## 2. 设置时间

点击主界面的“上班状态”或“下班状态”卡片，滚动选择小时和分钟并保存。
选择“每天”会每日执行；选择“仅周一至周五”会跳过周六、周日。

## 3. 锁屏与密码

到点后应用会点亮屏幕并请求系统解锁。存在指纹、密码或图案时，需要先
完成系统验证。应用不会读取、保存或绕过锁屏凭据。

只要企业微信尚未被无障碍服务确认进入前台，未完成通知就会持续保留。
点击该通知可以继续解锁并打开企业微信。

## 4. 息屏测试

1. 点击“立即传送至企业微信”，确认企业微信能够正常打开。
2. 返回 `Check in`，点击“安排 1 分钟息屏试跑”。
3. 立即返回桌面并锁屏。
4. 到点后完成解锁，确认企业微信进入前台。
5. 确认未完成通知在企业微信出现后自动消失。

## 5. 检查更新

点击主界面底部“检查更新”。首次使用时填写公开 GitHub 仓库：

```text
owner/repository
```

发现新版本后下载 APK，按系统提示允许 `Check in` 安装未知应用，返回后
会继续打开系统安装器。长按“检查更新”可以更换仓库。

## 6. 常见问题

### 到点没有任何反应

检查精确闹钟、通知、无障碍服务、电池无限制和厂商自启动设置，然后重新
完成一次息屏测试。

### 有通知但没有打开企业微信

说明任务没有完成，常见原因是锁屏验证、屏保、厂商后台限制或企业微信
未安装。点击通知继续处理。

### 系统升级后失效

打开 `Check in`，重新处理“系统体检”中的橙色项目。部分系统升级会关闭
无障碍服务或恢复电池限制。

## 能力边界

`Check in` 只负责在设定时间打开企业微信，不会代替用户点击企业微信中的
“打卡”按钮。
